### What I learned

During this project, I deepened my understanding of CSS layouts by handling complex layout configurations:
- Using CSS Grid (`grid-template-columns: 1fr 1fr`) to separate the hero copy and the rating boxes.
- Leveraging `align-self` properties on individual child items to achieve the staggered, structural "stair-step" look across columns and rows.
